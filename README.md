# Portfolio Website - Paramith Bandara

A modern, responsive personal portfolio website showcasing projects, skills, and professional background. Built with vanilla HTML, CSS, and JavaScript for a lightweight, performant user experience.

## 🌟 Features

- **Responsive Design** - Fully responsive layout that works on all device sizes
- **Smooth Navigation** - Smooth scrolling navigation with active link highlighting
- **Interactive Elements** - Scroll animations for skills and project cards
- **Contact Form** - Functional contact form with validation
- **Modern Styling** - Gradient backgrounds and contemporary design patterns
- **Performance Optimized** - Lightweight vanilla JS implementation without external dependencies

## 📁 Project Structure

```
├── index.html      # Main HTML file with page structure
├── style.css       # Styling and responsive design
├── script.js       # JavaScript interactivity and animations
├── README.md       # Project documentation
└── ...
```

## 🚀 Getting Started

### Prerequisites
- A modern web browser (Chrome, Firefox, Safari, Edge)
- No build tools or dependencies required

### Installation & Usage

1. **Clone or download** this repository to your local machine
2. **Open** `index.html` in your web browser
3. That's it! The website will load and run immediately

## 🎨 Customization

### Update Personal Information
Edit `index.html` to update:
- Your name (replace "Paramith Bandara")
- Bio and professional title
- Skills and expertise
- Project descriptions and links
- Social media links

### Modify Styling
Edit `style.css` to customize:
- Color scheme via CSS variables (`:root` section)
- Fonts and typography
- Layout and spacing
- Animations and transitions
- Responsive breakpoints

### Add Interactivity
Edit `script.js` to enhance:
- Form handling and validation
- Scroll animations
- Navigation behavior
- Additional interactive features
